# KeenKeeper

A personal relationship management app to help you stay connected with the people who matter most.

## 🛠️ Technologies Used

- Next.js 14
- Tailwind CSS
- Recharts
- React Toastify
- localStorage (for data persistence)

## ✨ Key Features

- **Friend Tracking** — View all your friends with their contact status (on-track, almost due, overdue)
- **Quick Check-In** — Log a Call, Text, or Video interaction directly from a friend's detail page
- **Friendship Analytics** — Visualize your interaction history with a Pie Chart broken down by type

## 📦 Getting Started

```bash
npm install
npm run dev
```
